# wp-social-verify-oauth

> 🔐 Secure WordPress OAuth plugin for Instagram & TikTok account verification. Built to demonstrate production-ready compliance, platform review preparation, and secure token handling.

⚠️ **Portfolio Project**: This repository uses official sandbox/test modes. Real app credentials, user data, and production endpoints are excluded per Meta & TikTok Developer Terms of Service.
